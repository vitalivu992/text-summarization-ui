Text summarization UI
===

## Development

### Setup environment

Use node v10.x

1. Install rekit-studio: `npm install -g rekit-studio`
2. Start rekit: `rekit-studio -p 3333`
3. Open browser: http://localhost:3333

### Start project

Use node v14.x

1. `yarn install`
2. `yarn start`
3. Open browser: http://localhost:3100
